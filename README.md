# Global Sales & Customer Behavior Analysis Using SQL

# Project Overview:-
This project focuses on performing an **End-to-End SQL Business Analysis** on a **Large Global E-Commerce Dataset (200,000 records)**.  
The analysis helps understand **Sales Performance, Customer Behavior, Cancellations, and Regional Trends** across both **National (India) and International Markets**.

-----

# Tools & Skills:-
- SQL
- Joins
- Data Aggregation (SUM, COUNT, GROUP BY)
- CASE Statements
- Window Functions (RANK)
- Time-based Analysis
- Business KPI Analysis
- Data Cleaning & Validation

-----

# Datasets Used:-
Table Name: "Global_Sales_Transactions_Raw"

- Rows: 200,000  
- Columns: 13  
- Data Type: National + International E-commerce Transactions

# Column Name:
1) order_id
2) customer_id
3) country
4) city
5) order_date
6) order_amount
7) currency
8) payment_method
9) order_status
10) product_category
11) quantity
12) sales_channel
13) delivery_days

-----

# Business Objective:-
The business wants to understand:
- Which countries and cities generate the highest revenue.
- How customer purchasing behavior differs across regions.
- What factors contribute to cancellations and returns.
- Which products, payment methods, and channels perform best.
- How delivery performance impacts customer satisfaction.

-----

# Project Structure:-
- Analysis/ : SQL Queries for analysis and insights.
- Insights/ : Business findings and recommendations.
- Raw_Data/ : Contains Raw Global_Sales_Transactions_Raw.csv" datasets.

-----

# Analysis Performed
- Data Understanding and Validation.
- Revenue and Order KPI Analysis.
- Country-Wise and City-Wise Revenue Analysis.
- Monthly Sales Trend Analysis.
- Cancellation and Return Rate Analysis.
- Customer Segmentation based on Spending.
- Top Cities Per Country using Window Functions.
- Channel and Payment method Performance Analysis.

-----

# Key Outcomes:-
- Revenue is concentrated in a few major cities within each country.
- India leads in transaction volume, while international markets show higher average order value.
- Electronics and Fashion are the top-performing categories.
- Mobile App generates more orders compared to Website.
- Longer delivery times are associated with higher return rates.
- High-value customers have lower cancellation behavior.
